# EconTools.descriptive_stats — Function.

descriptive_stats(table::DNextTable,var::String,bycolumn::Tuple;weight="None")

Returns a new table containing the mean, standard deviation, 1st, 2nd and 3rd quantiles of variable var by a selection of tuple of columns bycolumns. Weights are optional and if specified, the function returns the weighted descriptive statistics.

For example descriptive_stats(data,"income",(:age,:gender),weight="sampling_weights") returns the descriptive statistics of income from table data by the columns name age and gender, weighted by the sampling weights sampling_weights

# EconTools.save_csv — Function.

save_csv(path::String,table::DNextTable)

Saves table as CSV at the specified path. For example save_csv("Folder\data.csv",t) saves the table t in the subfolder Folder under the name data.csv
